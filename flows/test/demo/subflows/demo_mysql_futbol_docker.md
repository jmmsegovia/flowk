# subflows.demo_mysql_musicbrainz_docker

Detailed Description
MySQL flow that persists MusicBrainz artist data using Docker, FOR, PARALLEL, and EVALUATE actions. Primary actions: MySQL operations, Docker container management, assertions and conditional logic, iterations/loops, parallel execution, console output, controlled waits, variable configuration.

Requirements
- Docker installed and daemon running.
- Reachable MySQL instance and valid credentials.

# demo_complex_flow

Detailed Description
Complex demo that combines HTTP assertions, Dockerized MySQL persistence, and shell-based PGP encryption via subflows. Imports 3 subflow(s): demo_musicbrainz_artist_asserts, demo_mysql_futbol_docker, demo_pgp_password. Primary actions: MySQL operations, Docker container management, assertions and conditional logic, iterations/loops, HTTP/HTTPS requests, parallel execution, PGP encryption/decryption, console output, controlled waits, variable configuration.

Requirements
- Docker installed and daemon running.
- Reachable MySQL instance and valid credentials.
- PGP keys/aliases configured as required by the flow.
- HTTP connectivity to the configured endpoint(s).

# musicbrainz_json_asserts

Detailed Description
Flow generated from a Bruno test to validate the MusicBrainz JSON response (subset compatible with EVALUATE). Imports 1 subflow(s): docker_http_mock_musicbrainz. Primary actions: Docker container management, assertions and conditional logic, HTTP/HTTPS requests, controlled waits, variable configuration.

Requirements
- Docker installed and daemon running.
- HTTP connectivity to the configured endpoint(s).

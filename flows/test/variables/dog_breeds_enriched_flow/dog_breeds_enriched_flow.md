# dog_breeds_enriched_flow

Detailed Description
Reference flow that interacts with the Dog API and demonstrates advanced use of VARIABLES, PRINT, and EVALUATE. Imports 1 subflow(s): docker_http_mock_dog_api. Primary actions: Docker container management, assertions and conditional logic, HTTP/HTTPS requests, console output, controlled waits, variable configuration.

Requirements
- Docker installed and daemon running.
- HTTP connectivity to the configured endpoint(s).
